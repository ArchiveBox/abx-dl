REST API

Submit a download

POST /api/download Submit a URL for downloading. Returns session info with ID. 
curl -X POST {{ request.host_url }}api/download \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com", "plugins": ["title", "screenshot", "pdf"]}'

Check session status

GET /api/session/<id> Get session status, progress records, and file listing. 
curl {{ request.host_url }}api/session/a1b2c3d4e5f6

List all sessions

GET /api/sessions List all active sessions (newest first). 
curl {{ request.host_url }}api/sessions

Download files

GET /download/<id>/<path> Download an individual result file.
GET /download/<id>/zip Download the entire session output as a zip archive. 
# Download a single file
curl -O {{ request.host_url }}download/a1b2c3d4e5f6/title/title.txt

# Download everything as zip
curl -O {{ request.host_url }}download/a1b2c3d4e5f6/zip

Example: full workflow

# 1. Submit a download
ID=$(curl -s -X POST {{ request.host_url }}api/download \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com"}' | jq -r '.id')

# 2. Poll for completion
while true; do
  STATUS=$(curl -s {{ request.host_url }}api/session/$ID | jq -r '.status')
  echo "Status: $STATUS"
  [ "$STATUS" != "running" ] && [ "$STATUS" != "starting" ] && break
  sleep 3
done

# 3. Download results
curl -O {{ request.host_url }}download/$ID/zip
